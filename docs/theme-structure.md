\# Halo 主题文件结构分析



\## 项目根目录

halo-blog/



\## 核心目录结构

├── templates/ # 页面模板目录

│ ├── index.html # 首页模板

│ ├── post.html # 文章详情页

│ ├── category.html # 分类页

│ └── tag.html # 标签页

├── static/ # 静态资源

│ ├── css/ # 样式文件

│ ├── js/ # 交互脚本

│ └── images/ # 图片资源

├── docs/ # 项目文档

├── README.md # 项目说明

└── theme.yaml # 主题配置文件



\## 首页文件位置

`templates/index.html` 是博客首页，后续改造卡片布局、轮播图都在这里修改。
