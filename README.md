## 开发环境运行

```bash
# pnpm@7.0.0+
npm install -g pnpm
```

```bash
pnpm install 
```

```bash
# build packages
pnpm build:packages
```

```bash
pnpm dev
```

## 生产构建

```bash
pnpm build
```
